Métodos y Funciones de JAVASCRIPT

Funciones Math

Math.abs()

Se usa para obtener el valor absoluto de un número.

EJEMPLO

Math.round()

Se usa para obtener el valor de un número redondeado al entero más cercano.

EJEMPLO

Math.ceil()

La función Math.ceil() devuelve el entero mayor o igual más próximo a un número dado.

EJEMPLO

Math.floor()

Devuelve el máximo entero menor o igual a un número.

EJEMPLO

Math.exp()

La función Math.exp() devuelve e^x, donde x es el argumento, y es El número de Euler (también conocido como la constante de Napier), la base de los algoritmos naturales.

EJEMPLO

Math.log()

Esta función es equivalente a ln(x) en matemáticas. Si el número es negativo, se devuelve NaN.

EJEMPLO

Math.random()

Retorna un número aleatorio dentro del rango [0, 1). Esto es, desde el 0 (Incluido) hasta el 1 pero sin incluirlo (excluido), el cual se puede escalar hasta el rango deseado.

EJEMPLO

Métodos Array

Push()

Añade uno o más elementos al final de un array y devuelve la nueva longitud del array.

EJEMPLO

Pop()

Elimina el último elemento de un array y lo devuelve. Este método cambia la longitud del array.

EJEMPLO

Método reverse

Reverse

Invierte el orden de los elementos de un array in place. El primer elemento pasa a ser el último y el último pasa a ser el primero.
